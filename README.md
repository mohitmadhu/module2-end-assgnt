# module2-end-assgnt
Sign-Up form with Validation

Features:
1. Real Time Validation
2. Error Message Handling
3. Responsive

Tools:
HTML5, CSS, Javascript

Functionality:

Each input field is validated individually through each functions namely - NameValidation(), EmailValidation(), PasswordValidation(), ConfirmpasswordValidation(), TermsValidation().
Each function returns a boolean value (true or false) based on whether the field passes validation.
When the Submit button is clicked, the script checks whether all validation functions return true.
Only if every field is valid, the form is submitted successfully.
